# 使い方ガイド：目標 → 月別TODO → 週別TODO

## 全体の流れ

```
1. Goal(目標)を作成
   ↓
2. Goal の「分解」ボタン → 月別TODO作成
   ↓
3. 月別TODO の「分解」ボタン → 4つの週別TODOを作成
   ↓
4. 📅 今週のTODO セクションで進捗を確認
   ↓
5. タスクを「今日のTODO」に移動して実行
```

---

## ステップ 1: Goal(目標)を作成

### どこから作成するか
- 画面上部の **[+ Goal追加]** ボタンをクリック
- または **Goals** タブから新規作成

### 設定項目
- **タイトル**: 目標の名前（例：「技術士試験対策」）
- **カテゴリ**: vfx / english / investment / fitness / personal から選択
- **期間**: annual / monthly / weekly から選択 → **monthly** を推奨
- **目標値**: 数値を入力（例：60）
- **単位**: 時間 / 回 など（例：時間）
- **開始日・終了日**: 月の開始日から終了日を指定

### 例
```
タイトル: 技術士試験対策
カテゴリ: investment
期間: monthly
目標値: 60 単位: 時間
開始日: 2026-04-01
終了日: 2026-04-30
```

---

## ステップ 2: Goal を分解 → 月別TODO作成

### どこから実行するか
1. **Goals** タブを開く
2. 作成した Goal を探す
3. Goal の **[分解]** ボタンをクリック

### 設定内容
Goal を分解すると以下の画面が出ます：

```
各カテゴリに時間を配分：
- VFX: 20 時間
- English: 15 時間
- 技術士: 25 時間
```

**カテゴリ別配分**を入力して、「分解」をクリック

### 結果
✅ 月別TODO が作成される
- タイトル: 「技術士試験対策」
- 内容: カテゴリ別配分と詳細タスク一覧が記載
- 状態: **是_monthly_base = true**（月別フラグ）
- **期限: なし**（毎月作成されるため）

---

## ステップ 3: 月別TODO を分解 → 4つの週別TODOを作成

### どこから実行するか
1. **TODO** タブを開く
2. **📅 今週のTODO** タブをクリック
3. マスターリスト下部で、月別TODO を探す
4. TODO の **[分解]** ボタンをクリック

### 何が起こるか
✅ 4つの週別TODO が自動生成されます：

```
(W1) 技術士試験対策 (W1)
  - 優先度: 低（簡単なタスク）
  - 期限: 4月7日（月曜）

(W2) 技術士試験対策 (W2)
  - 優先度: 中
  - 期限: 4月14日（月曜）

(W3) 技術士試験対策 (W3)
  - 優先度: 中
  - 期限: 4月21日（月曜）

(W4) 技術士試験対策 (W4)
  - 優先度: 高（難しいタスク）
  - 期限: 4月30日（水）月末日）
```

**優先度を段階的に上げる**ことで、
- W1: 簡単なタスクから始める
- W4: 難しいタスクで応用力を身につける

という構成になっています。

## ステップ 4: 📅 今週のTODO で進捗確認

### UI の見方

```
┌─────────────────────────────────────┐
│   📅 今週のTODO (0/4)               │
│   ████░░░░░░░░░░░░░░░░░░░░░░  0%    │
│                                     │
│  🗓️ 2026-04-07                      │
│  ☐ 技術士試験対策 (W1)              │
│  ☐ VFX作業 (W1)                    │
│                                     │
│  🗓️ 2026-04-14                      │
│  ☐ 技術士試験対策 (W2)              │
│                                     │
│  📋 マスターリスト                   │
│  ☐ その他の全タスク...              │
└─────────────────────────────────────┘
```

### チェック方法
1. **進捗バー**: 完了したタスク数 / 全タスク数
2. **期限でグループ化**: 同じ期限のタスクが まとめて表示
3. **「今日へ」チェックボックス**: タスクを「今日のTODO」に移動

## ステップ 5: 「今日のTODO」で実行

### どこから見るか
1. **TODO** タブを開く
2. **📅 今日のTODO** タブをクリック

### タスク追加方法
### 方法1: 「今週のTODO」から移動
1. 「今週のTODO」タブで、実行するタスクを探す
2. タスク右の **「今日へ」チェックボックス** をクリック
3. ✅ 「今日のTODO」に移動

### 方法2: 新規タスクを追加
1. **「今日のTODO」タブ内の「+ 今日のタスク追加」** をクリック
2. タスク内容を入力
3. 保存 → 「今日のTODO」に追加

### 進捗表示
```
┌─────────────────────────────────────┐
│  📅 今日のタスク    完了: 2/5       │
│  ████████░░░░░░░░░░░░░░░░░░  40%    │
│                                     │
│  ☐ 技術士（通勤）1時間             │
│  ☑ VFX作業 2時間 [完了 14:30]      │
│  ☐ Duolingo 15分                   │
│  ☑ シャワー 15分 [完了 21:15]      │
│  ☐ 瞑想 30分                       │
│                                     │
│  [🎯 集中モード] [🔄 全て戻す]      │
└─────────────────────────────────────┘
```

### 操作
- **☑ チェック**: タスク完了
- **優先度**: 高/中/低 を選択
- **時間帯**: 朝🌅 / 昼☀️ / 夜🌙 を選択
- **🎯 集中ボタン**: そのタスクに集中する（シングルフォーカス）

## よくある質問

### Q1: 月別TODO は毎月自動で作成されるか？
**A**: Vercel の無料プランでは Cron が動作しないため、手動で「分解」ボタンを押す必要があります。
月初めに手動で分解するか、月替わり時に管理画面で実行してください。

### Q2: 週別TODO の優先度が自動で低→高になるのはなぜ？
**A**: 学習理論に基づいて、最初は簡単なタスク（W1, 優先度低）から始めて、徐々に難しいタスク（W4, 優先度高）に進むように設計しています。
- W1: 基礎知識の習得（優先度 低 = 時間がなくても OK）
- W4: 実践応用（優先度 高 = 必ずやる）

### Q3: 「今週のTODO」と「今日のTODO」の違いは？
**A**:
- **今週のTODO**: 全タスク一覧（マスターリスト）。期限で分類。進捗確認用。
- **今日のTODO**: 本日実行するタスク。時間帯別に管理。実行・完了追跡用。

### Q4: マスターリストに戻す方法は？
**A**:
「今日のTODO」タブの **[🔄 全て戻す]** ボタンをクリックすると、今日のTODOがすべてマスターリストに戻ります。
